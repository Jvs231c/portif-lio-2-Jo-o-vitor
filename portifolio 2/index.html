Efeito Mona Lisa: Uma Interatividade Criativa com CSS

O projeto aplica um efeito interativo sobre a imagem da Mona Lisa utilizando apenas HTML e CSS. A ideia é fazer com que, ao passar o mouse sobre a imagem, ela comece a se mover e reagir com diferentes animações. O efeito foi desenvolvido para criar uma experiência visual interessante e divertida, combinando a arte com a tecnologia.

Usando o CSS, são aplicadas animações de transformação e transição que alteram o comportamento da imagem. É uma forma divertida de demonstrar como simples interações com o CSS podem resultar em efeitos complexos, trazendo mais vida a uma página.

Jogo de Quente e Frio: Adivinhação Interativa com JavaScript

Este projeto é um jogo simples de adivinhação onde o usuário tenta adivinhar um número aleatório gerado pelo computador. O jogo responde com dicas como "quente" ou "frio", dependendo da proximidade do número escolhido em relação ao número correto.

O jogo é desenvolvido utilizando HTML, CSS para a interface e JavaScript para a lógica do jogo, incluindo a geração do número aleatório e a interação com o usuário. A interatividade e os elementos de feedback tornam o jogo divertido e fácil de jogar.
